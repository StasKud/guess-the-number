# Угадай число

## Описание проекта
Проект представляет собой консольное приложение, в котором пользователь пытается угадать число, загаданное компьютером. Программа предоставляет подсказки, указывая, больше или меньше введенное число по сравнению с загаданным, и ведет запись всех попыток пользователя в файл `log.txt`.

## Основные функции

### main()
Главная функция, которая запускает игру.

**Входные параметры:**
- Нет.

**Выходные параметры:**
- Нет.

**Описание:**
1. Генерирует случайное число от 1 до 100, которое пользователь должен угадать.
2. Инициализирует переменные для хранения максимального количества попыток, текущего количества попыток и списка введенных чисел.
3. В цикле `while` до достижения максимального количества попыток:
   - Запрашивает у пользователя ввод числа.
   - Обрабатывает исключения при вводе некорректных данных.
   - Сравнивает введенное число с загаданным и выводит соответствующие подсказки.
   - Записывает результат игры в файл `log.txt`.
4. Завершает игру, если число угадано или количество попыток исчерпано.

## Пример использования

1. Запустите программу:

    ```sh
    python main.py
    ```

2. Следуйте инструкциям на экране. Введите числа в ответ на запросы программы.
3. После завершения игры, откройте файл `log.txt`, чтобы просмотреть историю ваших попыток.

## Формат файла `log.txt`

Файл `log.txt` содержит следующую информацию:
- Загаданное число.
- Количество попыток.
- Введенные числа.
- Результат (угадано число или нет).
